Документ о системе как компиляторе знаний.

Pipeline:

```
Ontology + Data → Graph → Projection → ViewModel → UI
```

Содержит аналогии:

- ontology = язык знаний
- graph = semantic AST
- projection = intermediate representation
- ViewModel = target representation
- UI = runtime observation

Главная формула:

```
Meaning Engine = knowledge compiler + runtime
```
